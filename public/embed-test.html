Dango Header 埋め込みテスト

このページは <dango-header> Web Component の動作確認用です。 実際の外部アプリ（カレンダー・シェアリンク等）にこのヘッダーを埋め込んだ場合の 表示・挙動をシミュレートしています。

1. 埋め込みコード

各アプリの layout.tsx に以下の2行を追加するだけ：

<dango-header active-tool="calendar"></dango-header>
<Script src="https://dango-streamverse.vercel.app/dango-header.js" strategy="afterInteractive" />

2. 動作テスト

active-tool 属性の切り替え

ボタンをクリックすると active-tool 属性が動的に変更され、 デスクトップナビの対応リンクにアクティブインジケータ（光る下線）が表示されます。

スクロール検証

下にスクロールすると、デスクトップヘッダーの背景が透明からガラスモーフィズムに トランジションし、パディングが縮小します。

モバイル検証

ブラウザ幅を768px以下に縮めると、デスクトップヘッダーが非表示になり、 画面下部に「STREAMVERSE」フローティングポッドが出現します。 タップするとスライドアップドロワーが開きます。

Shadow DOM スタイル隔離

このページの CSS（body のフォントや色設定）が <dango-header> 内部のスタイルに影響を与えていないことを確認してください。 逆に、ヘッダーの CSS がこのページのレイアウトを崩していないことも確認します。

ログインボタン

デスクトップヘッダーの「ログイン」ボタン、またはモバイルドロワーの 「DANGO ID でログイン」ボタンをクリックすると、画面上部にトースト通知 「🍡 ログイン機能は近日公開予定です」が表示されます。
